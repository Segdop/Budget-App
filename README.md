# Budget App

A browser-based personal budget planner built with vanilla HTML, CSS, and JavaScript.

## Features
- Multiple income sources
- Expense tracking by category and owner
- Savings goals with target dates
- Investment projections with compound growth
- Disposable income allocation visualization
- LocalStorage persistence
- Export / Import backup support

## How to Run
Open `index.html` in a browser  
—or—  
Use VS Code Live Server
